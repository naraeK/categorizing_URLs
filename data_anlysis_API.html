def download_csv(file_csv, function, symbol=google, api_key=api_key):
    url = "https://www.alphavantage.co/query?function="+function+"&symbol="+symbol+"&apikey="+api_key+"&datatype=csv"
    urllib.request.urlretrieve(url, file_csv) # request a file by the url and save it to the name of file_loc

def filling_mean(dataset):
    null_info = dataset.isnull().sum()
    for k,v in null_info.items(): # k-keys; v-values of the null info. 
        if v != 0: # if there are missing values
            mean_key = dataset[k].mean()
            dataset[k] = dataset[k].fillna(mean_key) # fill up the empty value to the mean value
    return print("Data Cleaning Done.\n")

Tentative Conclusion¶

The daily and monthly based datasets are showing increasing patterns for Google and Microsoft as time goes by. The interesting thing is that both companies have very similar patterns as a positive proportional to each other. As we saw, the shorter term of the data seems more robust to predict in future. On the other hand, the longer term is dispersed around which implies higher variances so that the prediction can get a wider error range than the shorter one. For example, the daily basis dataset would be a good choice to predict if the stock will increase or decrease. However, as much of the time scale as the monthly basis data is, we would be allowed to predict further future with the monthly data instead of the daily data. As a result, I found Google and Microsoft stocks high potential to get even higher from the trends.

Nonetheless, we could expect more profits by Google than by Microsoft as long as we find the increasing proportion/ratio nearly the same between them. For instance, 1000 by 10% increasing is 1100 and 100 by 10% is 110. Therefore, 1000 gets 90 (100-10) more even if 10 times of 100 will generate the same result as 1000 does. In this data, Google stock seems 12 times higher value than Microsoft's. Moreover, the ratio is about double within 3 years, 67% increase per year during last three years.

For the better prediction, I suggest getting more information about the past years. If possible, it will be also helpful with other IT companies stock information to understand the trend for the same industry. Lastly, the stocks of their joint corporations should be a good indicator as well.
